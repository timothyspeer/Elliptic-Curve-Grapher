About

Enter values for a and b to graph the elliptic curve given by the equation y2 = x3 + ax + b

The curve will be plotted on the interval [-x range, x range]. The variables a and b can take values in the interval [-100, 100] with an accuracy of one decimal point. The variable x range can take values in [1, 500] in increments of 1. If you enter values for a and b which make the curve singular then the graph for the singular curve will still be plotted.
